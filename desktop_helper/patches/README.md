# Patch Series

Place desktop-helper patch files here.

Expected workflow:

1. check out the upstream Zotero revision pinned in `../metadata.json`
2. maintain a small patch series that contains only the helper-specific delta
3. update `metadata.json` whenever the upstream revision changes

This directory intentionally starts empty. The repo no longer treats a vendored Zotero snapshot as the source of truth for helper modifications.
